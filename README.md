# Cadastro de Usuário

Projeto desenvolvido para praticar conceitos de Front-end utilizando HTML, CSS e JavaScript puro.

O sistema simula um cadastro de usuário dividido em duas etapas, contendo validações de formulário, máscaras de campos e integração com a API ViaCEP.

---

## 📸 Demonstração

O projeto possui:

- Tela de cadastro inicial
- Tela complementar de cadastro
- Validação de formulários
- Máscaras para CPF, telefone e CEP
- Busca automática de endereço pelo CEP
- Responsividade para diferentes tamanhos de tela

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- API ViaCEP

## 🌐 API utilizada

### ViaCEP

Utilizada para realizar a busca automática do endereço através do CEP informado.

Site oficial: https://viacep.com.br/
